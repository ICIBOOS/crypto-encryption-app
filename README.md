# 🔐 Aplikasi Enkripsi & Dekripsi Digital

Aplikasi web modern untuk enkripsi dan dekripsi data dengan dua mode: **ECB** dan **CBC**.

## ✨ Fitur

- ✅ **Mode ECB**: Substitusi → Permutasi → Rotasi
- ✅ **Mode CBC**: Feistel Network → Substitusi → Permutasi → XOR Chaining
- ✅ **Input Fleksibel**: Text atau upload file .txt
- ✅ **Kunci Dinamis**: Manual atau generate otomatis
- ✅ **Output Formats**: HEX, Binary, Base64
- ✅ **Proses Transparan**: Lihat detail setiap langkah enkripsi
- ✅ **Export**: Copy atau download hasil
- ✅ **Responsive UI**: Desktop & mobile friendly

## 🚀 Cara Menggunakan

1. **Pilih Mode**: Klik Mode ECB atau CBC
2. **Input Text**: Ketik minimal 3 karakter atau upload file .txt
3. **Masukkan Kunci**: Gunakan kunci custom atau generate otomatis
4. **Enkripsi**: Klik tombol Enkripsi
5. **Hasil**: Copy atau download hasil enkripsi

## 📋 Ketentuan Proyek

✅ Tidak menggunakan library enkripsi yang sudah ada
✅ Menggunakan minimal 2 metode dasar kriptografi per mode
✅ Mode ECB dan CBC memiliki metode yang berbeda
✅ Input: text (minimal 3 karakter) atau file .txt
✅ Output: HEX, Binary, atau Base64
✅ UI modern dan responsive
✅ Dapat diakses orang lain (online atau lokal)
